# HabitOS Website

This is the marketing website for HabitOS.

## Auto-Deploy Setup

This site is connected to Netlify for automatic deployment.

**To update the live site:**
```bash
git add .
git commit -m "Update website"
git push origin main
```

Netlify will automatically deploy within 1-2 minutes.

## Files
- `index.html` - Main landing page
- `payment.js` - Payment integration (Razorpay/Gumroad)
- `style.css` - Styling
- `script.js` - Interactive elements
- `HabitOS-Setup.exe` - Windows installer
